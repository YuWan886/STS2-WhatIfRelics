# What If Relics

将 YuWanCard 中的"假如"遗物系列拆分为独立模组，开局可选择改变整局游戏规则的遗物。

## 功能

- **开局选择**：在每局游戏开始时选择一个"假如"遗物，改变这一局的游戏规则
- **细粒度替换范围**：可分别控制初始卡组、卡牌奖励、遗物奖励、药水奖励、宝箱、商店等场景是否生效
- **多人同步**：宿主权威的配置同步，确保多人游戏中所有玩家的规则一致

A standalone extraction of the "What If" relic series from YuWanCard. At the start of each run, choose a relic that bends the rules of the entire game.

## Features

- **Run-Start Selection**: Pick a "What If" relic before every run to reshape the rules
- **Fine-Grained Replacement Scope**: Independently toggle replacements for starting deck, card rewards, relic rewards, potion rewards, treasure chests, and shops
- **Multiplayer Sync**: Host-authoritative config sync keeps rules consistent across all players
